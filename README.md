# AI-Agent-Boilerplate

This is a boilerplate code to your ai agent, i have customized it a little bit to help me in my daily pentesting and red team exercises
You can modifiy the system prompt and give it any context you want.

## How to run
1. copy the git repo
2. create a virtual environment using `python -m venv .myenv` and activate it `source .myenv/bin/activate`
3. install the dependencies using `pip install -r requirments.txt`
4. make a .env file and add `GOOGLE_API_KEY=yourapikey123`
5. run the main file `python main.py` or `.myenv/bin/python main.py`
